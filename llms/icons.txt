# BlazorUI Icons - LucideIcon Component

The BlazorUI.Icons package provides access to 1,640+ beautiful, customizable Lucide icons as Blazor components.

---

## Installation

```bash
dotnet add package BlazorUI.Icons
```

## Setup

Add to your `_Imports.razor`:

```razor
@using BlazorUI.Icons
```

---

## Basic Usage

Render icons using the `<LucideIcon>` component with the `Name` parameter:

```razor
<LucideIcon Name="icon-name" />
```

**Icon names use kebab-case** (e.g., `"camera"`, `"chevron-down"`, `"arrow-right"`).

### Examples

```razor
@* Basic icons *@
<LucideIcon Name="house" />
<LucideIcon Name="settings" />
<LucideIcon Name="user" />

@* Icons with custom size *@
<LucideIcon Name="search" Size="20" />
<LucideIcon Name="bell" Size="24" />

@* Icons with custom color *@
<LucideIcon Name="heart" Color="red" />
<LucideIcon Name="star" Color="#FFD700" />

@* Icons with CSS classes (Tailwind) *@
<LucideIcon Name="check" Class="text-green-500" />
<LucideIcon Name="x" Class="text-red-500" />
```

---

## API Reference

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Name` | `string` | **Required** | Icon name in kebab-case (e.g., "camera", "chevron-down") |
| `Size` | `int` | `24` | Icon size in pixels (width and height) |
| `Color` | `string` | `"currentColor"` | CSS color value (hex, rgb, named color, or "currentColor") |
| `StrokeWidth` | `double` | `2.0` | SVG stroke width (line thickness) |
| `Class` | `string` | `""` | Additional CSS classes to apply |
| `AriaLabel` | `string` | `null` | Accessibility label for screen readers |
| `AdditionalAttributes` | `Dictionary<string, object>` | `null` | Any additional SVG attributes |

### Examples

```razor
@* Full parameter usage *@
<LucideIcon
    Name="settings"
    Size="20"
    Color="#3B82F6"
    StrokeWidth="1.5"
    Class="rotate-90"
    AriaLabel="Settings icon" />
```

## Finding Icon Names

### Online Resources

1. **Lucide Icon Library**: https://lucide.dev/icons
   - Browse all 1,640+ icons
   - Search by keyword
   - Copy icon names directly

2. **Icon Name Format**
   - All icon names use **kebab-case**
   - Examples: `chevron-down`, `arrow-right`, `user-plus`
   - **NOT**: `ChevronDown`, `arrowRight`, `UserPlus`

### Common Naming Patterns

```razor
@* Directional icons: name-direction *@
<LucideIcon Name="arrow-left" />
<LucideIcon Name="arrow-right" />
<LucideIcon Name="chevron-up" />
<LucideIcon Name="chevron-down" />

@* Action icons: verb or verb-noun *@
<LucideIcon Name="edit" />
<LucideIcon Name="save" />
<LucideIcon Name="log-out" />
<LucideIcon Name="user-plus" />

@* Status icons: noun-state *@
<LucideIcon Name="check-circle" />
<LucideIcon Name="x-circle" />
<LucideIcon Name="alert-triangle" />

@* Numbered variants *@
<LucideIcon Name="loader" />
<LucideIcon Name="loader-2" />
<LucideIcon Name="share" />
<LucideIcon Name="share-2" />
```

```razor

<LucideIcon Name="heart" Class="text-red-500" />
<LucideIcon Name="star" Class="text-yellow-400" />
<LucideIcon Name="check" Class="text-green-600" />

@* Animation *@
<LucideIcon Name="loader-2" Class="animate-spin" />
<LucideIcon Name="bell" Class="animate-bounce" />

@* Hover effects *@
<LucideIcon Name="heart" Class="hover:text-red-500 transition-colors" />
```

### Using Custom CSS

```css
/* Custom icon styles */
.icon-primary {
    color: var(--primary);
}

.icon-destructive {
    color: var(--destructive);
}

.icon-lg {
    width: 32px;
    height: 32px;
}

.icon-spin {
    animation: spin 1s linear infinite;
}

@keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}
```

```razor

```

## Accessibility

### Adding ARIA Labels

For icons that convey meaning (not just decoration), add `AriaLabel`:

```razor
@* Button with icon only - needs aria-label *@
<Button Size="icon" AriaLabel="Close">
    <LucideIcon Name="x" Size="20" AriaLabel="Close" />
</Button>

@* Icon with text - no aria-label needed *@
<Button>
    <LucideIcon Name="save" Size="16" />
    Save
</Button>
```

### Decorative vs. Meaningful Icons

```razor
@* Decorative (icon + text) - no aria-label needed *@
<Button>
    <LucideIcon Name="download" Size="16" />
    Download
</Button>

@* Meaningful (icon only) - aria-label required *@
<Button Size="icon">
    <LucideIcon Name="download" Size="20" AriaLabel="Download file" />
</Button>

@* Status indicator - add context *@
<div class="flex items-center space-x-2">
    <LucideIcon Name="check-circle" Size="16" Class="text-green-500" AriaLabel="Success" />
    <span>Operation completed</span>
</div>
```

## Performance Tips

1. **Icon Size**: Use appropriate sizes for context
   - Small UI elements: 12-16px
   - Buttons: 16-20px
   - Headers: 20-24px
   - Hero sections: 32px+

2. **Color**: Prefer `currentColor` or CSS classes over inline colors for better themability

3. **Reusability**: Create wrapper components for frequently used icon combinations

```razor
@* IconButton.razor - Reusable icon button component *@
<Button Size="icon" Variant="@Variant" @onclick="OnClick">
    <LucideIcon Name="@IconName" Size="20" AriaLabel="@AriaLabel" />
</Button>

@code {
    [Parameter] public string IconName { get; set; } = "";
    [Parameter] public string Variant { get; set; } = "ghost";
    [Parameter] public string AriaLabel { get; set; } = "";
    [Parameter] public EventCallback OnClick { get; set; }
}
```

## Troubleshooting

### Icon doesn't appear

**Cause:** Icon name is incorrect or not in kebab-case

**Solution:**
- Check icon name at https://lucide.dev/icons
- Ensure name is kebab-case: `"chevron-down"` not `"ChevronDown"`
- Verify BlazorUI.Icons package is installed

### Icon is too large/small

**Cause:** Size parameter or CSS classes affecting dimensions

**Solution:**
```razor
@* Use Size parameter *@
<LucideIcon Name="home" Size="20" />

@* OR use Tailwind classes *@
<LucideIcon Name="home" Class="w-5 h-5" />

@* Don't mix both - choose one approach *@
```

### Icon color doesn't change

**Cause:** Using `Color` parameter with CSS color classes

**Solution:**
```razor
@* Option 1: Use Color parameter *@
<LucideIcon Name="heart" Color="red" />

@* Option 2: Use CSS class (preferred for theming) *@
<LucideIcon Name="heart" Class="text-red-500" />

@* Don't use both simultaneously *@
```
