@* Set Id, then point a <label for="..."> at it to open the file dialog from your own
   control. The label works even when the dropzone itself is hidden, so the upload can be
   driven from somewhere else entirely — an attach button in a chat composer, say. *@

<label for="chat-attachment-upload" class="...">
    <LucideIcon Name="paperclip" Class="h-4 w-4" />
    Attach files
</label>

<BbFileUpload Id="chat-attachment-upload" Multiple="true" />

@* Without Id, the input still gets a generated, unique id — you just can't target it. *@
<BbFileUpload />

@* Unmatched attributes are applied to the underlying <input type="file">, so form and
   accessibility attributes reach the element that actually holds the files. Class styles
   the container instead. *@
<BbFileUpload Id="documents" name="documents" aria-label="Upload documents" Class="max-w-md" />
